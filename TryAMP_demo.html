Introduction to Aeris.js beta Weather Map

Aeris Maps Platform (AMP) Animation - FREE version

*Note: This demo is for the free version with features restriction; hence, some map layers are not available unless upgrade subscription.


This map shows the radar + infrared satellite for the past 24 hours in a 30 seconds timelapse animation (Sault Ste. Marie, Ontario + Michigan).

  1. Setup Aeris.js Weather:
    • Link aerismaps-animation.min.js file from Aeris Maps Animator CDN
    • Link aerismaps-animation.css file from Aeris Maps Animator CDN
    • Sign up a FREE AerisWeather API Developer account
    • Get an Aeris API key, will need yours coding this webmap to work.

  2. Configure Aeris API key

  3. *Choose the Aeris Map Tiles layers to overlay:
    • Radar -USA updates per 6min.
    • Infrared Satellite -North America, updates per 30 min.

  4. You have created your first AerisWeather map with temporal animation :) !

  5. Learn more at Aeris Maps Platform (AMP).
    • AerisWeather Animation
    • Support Help Center
    • Aeris Maps Animiation on GitHub

  6. If confuse, see the Intro to Aeris.js Weather Map demo.
