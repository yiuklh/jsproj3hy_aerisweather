Intro: Steps to create an Aeris.js beta Weather Map with Leaflet v0.7.7

Initial setup, Add weather map tiles layers, Adjust layer opacity (transparency)

Note: The "Esri World Imagery" was set for initial display, but Leaflet default basemap is actually the "OSM".


This demo will adding a radar layer onto a Leaflet.js map.

  1. Setup Aeris.js Weather from an existing Leaflet.js html.
    • Link Aeris.js: the mapping library for Leaflet
    • Sign up a FREE AerisWeather API Developer account
    • Get an Aeris API key, will need yours coding this webmap to work.

  2. Create a Leaflet.js map.

  3. Configure Aeris API key

  4. Add the Aeris Map Tiles Layers:
    • Radar -USA, updates per 6min.
    • Infrared Satellite -North America, updates per 30 min.
    • Current Temperatures

  5. Adjust layer opacity (transparency) for the infrared satellite layer.

  6. You have created your first AerisWeather map :) !

  7. Learn more at Aeris Maps Platform (AMP).

    • Aeris.js beta on GitHub
