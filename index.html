Welcome to Intro Aeris.js beta Weather Map (for Beginners)

Steps to create a basic weather map with Leaflet v0.7.7

  1. Setup Aeris.js Weather from an existing Leaflet.js html.
  2. Create a Leaflet.js map < http://leafletjs.com/index.html >.
  3. Get and Configure an Aeris API key.
  4. Add the Aeris Map Tiles Layers: Radar, Infrared Satellite
  5. Adjust layer opacity for the Aeris layers set in Step 4.
  6. You have created your first weather map with Aeris.js! :)
  7. Learn more at Aeris Maps Platform (AMP) < http://www.aerisweather.com/support/docs/aeris-maps/ >

Preface:

After my difficulties had with setting up an Aeris.js weather map, that's a basic one just to get started! Often the available documentations from GitHub or developer websites are considerd as advanced materials, which I usually can't understand them as a complete beginner. Some codes were even outdated and no longer work when attempted the codings. So I decided to compile a much simpler tutorial targeting for the beginners, whom they might have similar learning challenges.
